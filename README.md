# Game Night for Slack

Game Night for Slack is a Slack bot for easily hosting game nights.

## Installation

```bash
npm/yarn install
```

## Usage

### Production

```bash
npm/yarn run start
```

### Development

```bash
npm/yarn run dev
```

### Build

```bash
npm/yarn run build
```
